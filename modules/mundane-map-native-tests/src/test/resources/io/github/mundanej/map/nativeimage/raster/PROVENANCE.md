# G6-005 native raster fixtures

These five root-project BSD-3-Clause fixtures were authored for `mundane-java-map` on 2026-07-15.
They may be redistributed under the repository's BSD-3-Clause license. They are checked in as
fixed test evidence and are never regenerated by the build or native executable.

The PNG assets were generated with Python 3.12.3 standard-library `struct`, `zlib`, and `crc32`.
`png-affine-smoke.png` is a non-interlaced 4-by-4, 8-bit RGBA PNG with one filter-zero row per
source row and one IDAT. Its four 2-by-2 quadrants, in row order, are `(220,40,40,255)`,
`(30,180,80,255)`, `(40,90,220,255)`, and `(240,190,30,128)`.
`malformed-idat-crc.png` is an otherwise valid non-interlaced 1-by-1, 8-bit RGBA PNG. Its 13-byte
IDAT uses zlib level 1 with `Z_HUFFMAN_ONLY`; one bit of the IDAT CRC beginning at byte 54 was
flipped after construction.

`jpeg-affine-smoke.jpg` was generated with ImageMagick 6.9.12-98 Q16 using a 16-by-12 RGB canvas,
four constant 8-by-6 quadrants `(200,50,50)`, `(50,180,70)`, `(50,80,200)`, and `(220,190,40)`,
then `-strip -sampling-factor 4:4:4 -quality 95`. The checked bytes, not another encoder run, are
the test authority.

The world files are literal ASCII physical-line-order coefficients:

- `png-affine-smoke.pgw`: `20\n5\n4\n-18\n1000\n2000\n`
- `jpeg-affine-smoke.jgw`: `12\n2\n-3\n-10\n3000\n1000\n`
